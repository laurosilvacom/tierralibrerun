Working draft for a 2 to 3 page sponsor leave-behind.

## [Brand Name] x Tierra Libre Run

Tierra Libre Run is a BIPOC-led nonprofit trail access initiative building a clearer path into trail running for runners of color.

We fund race entries. We provide mentorship. We show up with real community support around race day.

Everything is organized around one concrete goal: helping runners of color get to the start line and feel like they belong there.

## Why Tierra Libre Run exists

Trail running talks a lot about freedom, resilience, and community. But for many runners of color, the path into the sport is still blocked by cost, access, and isolation.

The barriers are practical and cultural at the same time:

- entry fees are expensive
- gear and travel add up quickly
- many new runners do not see themselves reflected in trail running spaces
- getting to a first race often means doing it without mentorship or community

Tierra Libre Run exists to change that.

We are building the infrastructure that helps runners of color not just discover trail running, but actually enter it, stay in it, and lead in it.

## What Tierra Libre Run does

### Athlete Fund

We help cover race registration fees so cost is not the reason someone stays home.

### Mentorship

We pair runners of color with experienced mentors who can help them train, prepare, and move through the trail race experience with more confidence.

### Race-day community support

At select partner races, Tierra Libre Run shows up with in-person support, community presence, and a more welcoming path through race weekend.

## What a brand partnership makes possible

A sponsor partnership with Tierra Libre Run directly supports:

- funded race entries for runners of color
- mentorship and athlete preparation
- race-week communication and coordination
- community-building and storytelling
- on-the-ground race support
- core program operations that make the work sustainable

This is not abstract awareness work.

It is direct support tied to real runners, real races, and visible community outcomes.
